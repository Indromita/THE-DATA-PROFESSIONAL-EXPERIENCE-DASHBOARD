# Data Professional Survey Analysis

## Project Overview
I have analysed the insider perspective offered by data professionals through survey data collected via social media to highlight trends in demographics, work satisfaction and career paths among data professionals.

---

## Key Insights
* **Salaries:** Salaries varied depending on the roles but Data scientists stood out as the top earners whereas database developers drew the lowest salaries.
* **Industry Accessibility:** Is the data industry closed off? Around 42.7% think that it is not, portraying their neutrality. Yet, 24% do think its difficult to break into the field. So the data industry seems to be a complex field that might require some effort to enter. 
* **Job Satisfaction:** Scores indicated that the data professionals are moderately satisfied with their work life balance and salaries showing much room for improvement.

---

## Technical Workflow
### Steps taken to collect insights-
* **Data cleaning-** Removal of unnecessary columns and nulls and standardizing data, creating some calculated columns for better visualization.
* **Visualizing-** the survey breakdown through Gauge charts showing sentiments, tree maps showing the population diversity and donut charts visualising the complex opinions.
* **Interactivity-** Lastly, the charts were made interactive and user friendly.
